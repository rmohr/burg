Detailed Description

Interface for in memory databases.

every kind of user/password database which is loaded once at startup and is located in the server memory should inherit from this interface. As it provides a reload() method to allow the reload of the database, the database inherited from this interface should be threadsave. FileUserDB provides a reference implementation.

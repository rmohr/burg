Detailed Description

a file based user/password database

the structure of the database file is

users =
(
("user123", "password123")
("fritz", "03UdM/nNUEnErytGJzVFfk07rxMLy7h/OJ40n7rrILk=")
);

extension point for reload in a specific database implementation

the intention of the method is to allow the reload of the database when for example SIGHUP occures. As an application might be multithreaded, the whole derived class must be threadsave.
